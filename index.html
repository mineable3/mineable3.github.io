Technical Showcase

1 / FRC Team 1991's 2025 robot

A code base created to control an FRC robot. The robot is controlled by two state machines, one for translational movement and another for auxiliary control of scoring. It also implements the singleton design pattern for safety.

2 / Neural Networks

Available to view on Github is a neural network built from scratch, without the use of python libraries. While somewhat inefficient, it proved itself capable in a case study of it playing a version of Texas Hold'em against itself.

3 / Checkers

A very simple, but in my opinion charming game a checkers. Implemented with a CLI, it is a two player checkers game and plays the variation without double jumps.
